Getting started

The following instructions will tell you how you can begin making a page.

    Instructions:
  1. Open Brackets
  2. Create a new file and save it in a folder
  3. Create an Assets folder and CSS folder inside the folder
  4. Type <!DOCTYPE html> at the top
  5. Add a <title> tag to name the page
  6. Add a <body> tag to type everything you want the user to see in

Putting pictures in

Sometimes you may want to put pictures into your page. This is how you can do that.

    Instructions
  1. Move or copy the picture into the assets folder
  2. Put the picture into the page using the <img> tag. For example: <img src=koala.png>
  3. Add alt text so that if a browser can't show the image it can instead read it out. Ex: alt="A picture of a koala bear"
  4. Change the height or width to whatever you think will look right

Encoding a video

You may need to encode a video before you put it on a webpage. Here is one way to do that using Adobe Media Encoder.

    Instruction
  1. Open Adobe Media Encoder
  2. Add a source and click on it
  3. Change the settings to your liking and click OK
  4. Move the encoded video file to the assets folder
  5. Put the video in the page with the <video> tag and the <source> tag
